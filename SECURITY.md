# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in JQ.AI, please disclose it responsibly.

**Do not open a public GitHub issue.** Instead, email us at `security@juriquants.com`.

We will:
- Acknowledge receipt within **72 hours**.
- Provide an estimated timeline for a fix.
- Credit you publicly once the issue is resolved (if you wish).

## Safe Harbor

We provide a safe harbor for good-faith security researchers. If you follow our disclosure guidelines, we will not take legal action against you.

## Supported Versions

We currently support the latest stable release of JQ.AI. Security patches will be backported to the previous major version for 6 months after a new release.
